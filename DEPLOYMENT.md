# Udyam Registration Portal

## 🚀 Live URLs
- **Frontend**: https://uifron.vercel.app/
- **Backend**: https://udyam-mlhj.onrender.com

## 📁 Deployment

### Frontend (Vercel)
Deploy from `frontend/` directory with Vite framework

### Backend (Render)  
Deploy from `backend/` directory with Node.js runtime

## 🛠️ Development
```bash
npm run install:all
npm run dev
```
